# 🧘 마음수련 콘텐츠 자동화 봇 v2.0 (Vercel + Supabase 업그레이드 버전)

기존 Streamlit으로 제작된 마음수련 콘텐츠 자동화 봇을 최신 웹 환경에 맞추어 **React(싱글 페이지 앱)와 Tailwind CSS, Supabase 클라우드 데이터베이스** 기반으로 마이그레이션 및 업그레이드한 버전입니다.

비개발자 초보자도 쉽게 로컬에서 실행하고, 클라우드 데이터베이스에 데이터를 저장하며, 인터넷에 무료로 배포할 수 있도록 설계되었습니다.

---

## ✨ 주요 특징
- **설치 필요 없음**: Node.js나 복잡한 빌드 도구 설치 없이 브라우저에서 바로 열립니다.
- **수려한 디자인**: 현대적이고 반응형인 UI(Tailwind CSS)로 제작되었습니다.
- **클라우드 저장 기능**: Supabase와 연동하여 내가 작성하고 생성한 명상 일지를 클라우드 데이터베이스에 보관하고 불러올 수 있습니다. (연동 전에는 브라우저 내부 로컬 저장소에 안전하게 자동 저장됩니다.)
- **일괄 다운로드 및 복사**: 변환된 4종 소셜 콘텐츠(블로그, 명상 가이드, 인스타 캡션, 유튜브 대본)를 한 번에 텍스트 파일로 다운로드하거나 개별 복사할 수 있습니다.

---

## 🚀 로컬에서 실행하기 (2가지 방법)

### 방법 1: 더블클릭하여 바로 열기
1. `실습3 업그레이드` 폴더 내의 `index.html` 파일을 더블클릭합니다.
2. 사용 중인 웹 브라우저(Chrome, Edge, Safari 등)에서 즉시 실행됩니다.

### 방법 2: Python 로컬 서버로 열기 (권장)
컴퓨터에 Python이 설치되어 있다면 다음 명령어로 로컬 서버를 띄울 수 있습니다.
1. 터미널(PowerShell 또는 명령 프롬프트)을 엽니다.
2. `실습3 업그레이드` 폴더로 이동합니다.
3. 다음 명령어를 입력합니다:
   ```bash
   python -m http.server 8000
   ```
4. 웹 브라우저를 열고 `http://localhost:8000`에 접속합니다.

## 🗄️ Supabase 클라우드 데이터베이스 연동 가이드

데이터를 클라우드에 영구 저장하고 관리하려면 무료 클라우드 DB인 **Supabase** 설정을 진행해야 합니다.

### 1단계: Supabase 프로젝트 만들기
1. [Supabase 공식 홈페이지](https://supabase.com)에 접속하여 회원가입(GitHub 계정 로그인 가능)합니다.
2. 로그인 후 **New Project** 버튼을 누릅니다.
3. 프로젝트 정보를 입력합니다:
   - **Name**: `mindfulness-bot` (자유롭게 입력)
   - **Database Password**: 안전한 비밀번호 입력 (따로 메모해 두세요)
   - **Region**: `Seoul (ap-northeast-2)` (한국 지역 선택)
   - **Pricing**: `Free` 선택
4. **Create new project**를 누르고 프로젝트가 생성될 때까지 약 1~2분 기다립니다.

### 2단계: 데이터베이스 테이블 생성하기 (SQL 실행)
1. 프로젝트 생성이 완료되면 왼쪽 메뉴에서 **SQL Editor** (테이블 보양 아이콘 아래에 있는 `>_` 아이콘)를 클릭합니다.
2. **New Query** 버튼을 누릅니다.
3. 아래의 SQL 코드를 복사해서 입력창에 붙여넣습니다:

```sql
-- 1. 명상 콘텐츠 일지를 저장할 테이블 생성
create table records (
  id bigint generated by default as identity primary key,
  created_at timestamp with time zone default timezone('utc'::text, now()) not null,
  title text not null,
  input_text text not null,
  result_blog text not null,
  result_meditation text not null,
  result_instagram text not null,
  result_youtube text not null,
  category text default '마음수련'
);

-- RLS(행 보안 정책) 비활성화 (간단한 테스트 및 실습 목적)
alter table records disable row level security;
```

4. 우측 하단의 **Run** (또는 Ctrl + Enter) 버튼을 누릅니다. 성공적으로 완료되면 `Success. No rows returned` 메시지가 표시됩니다.

### 3단계: 웹 앱에 API Key 연결하기
1. Supabase 왼쪽 메뉴 맨 아래에 있는 **Settings** (톱니바퀴 아이콘)를 누릅니다.
2. **API** 메뉴를 클릭합니다.
3. 화면에 나타나는 **Project URL**과 **API Keys** 영역에서 `anon` `public` 키를 복사합니다.
4. 웹 브라우저에서 실행 중인 우리 웹 앱 왼쪽 하단의 **⚙️ Supabase Cloud DB 연동** 영역에 각각 붙여넣은 후 **데이터베이스 연결하기** 버튼을 클릭합니다.
5. 상단에 `Supabase에 연결되었습니다.` 알림이 뜨면 연동 완료! 이제 데이터를 저장하면 클라우드 DB에 실시간으로 기록됩니다.

## 🌐 Vercel에 무료로 웹 배포하기

이 웹 앱은 정적 파일(`index.html`)로만 이루어져 있어 Vercel을 통해 10초 만에 전 세계에 무료 배포할 수 있습니다.

1. [Vercel 공식 홈페이지](https://vercel.com)에 접속하여 GitHub 계정으로 가입/로그인합니다.
2. **Add New...** -> **Project**를 클릭합니다.
3. 본인의 GitHub 저장소 목록에서 `mindfulness-content-bot` (혹은 업그레이드용 저장소)를 선택하여 **Import**를 누릅니다.
4. **Configure Project** 화면이 나오면:
   - **Framework Preset**: `Other` 또는 `Create React App` (자동 감지된 기본값 유지)
   - **Root Directory**: `실습3 업그레이드` 폴더로 설정되어 있는지 확인 (만약 전체 저장소가 이 폴더 중심이면 루트 `./`로 설정)
5. **Deploy** 버튼을 클릭합니다.
6. 배포가 완료되면 대시보드에서 자동으로 발급된 무료 `https://...vercel.app` 웹 주소를 확인할 수 있습니다! 이제 스마트폰이나 다른 컴퓨터에서도 해당 주소로 접속해 사용할 수 있습니다.
